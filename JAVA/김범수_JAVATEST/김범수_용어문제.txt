10번 --
자바에서 상속받는 클래스를 "자식 클래스(서브 클래스)"라고 부르며, "extends" 키워드를 이용하여 상속을 선언한다. 상속받은 클래스에서 상속해준 클래스의 멤버를 접근할 때 "super" 키워드를 이용한다. 한편, 객체가 어떤 클래스의 타입인지 알아내기 위해서는 "instanceof" 연산자를 이용하며, 인터페이스로 클래스를 구현할 때는 "implements" 키워드를 이용하여 구현한다.
정답 : 자식 클래스(서브 클래스) / extends / super / instanceof / implements

상대 경로 지정 방법
1. 내 파일이 어디 있는지 ( HTML 폴더 )
2. 파일이 있는 폴더에서 내가 가려는 파일이 어디있는지
       ( HTML 폴더 -> CSS 폴더 -> CSS파일 )
현재 파일이 있는 폴더와 가려는 파일의 첫번째 폴더가 같으면  . ( 점 한개 ) 로 시작
./css/abc.css
. -> 현재 폴더를 지칭하는 키워드
----------------------------------------------------------------
1. 내 파일이 어디있느냐 ( HTML -> CSS 폴더 )
2. 내가 가려는 파일이 어디있는지
( HTML 폴더 -> images폴더 -> bg.jpg )
현재 파일이 있는 곳에서 상위 폴더로 올라가야하면
.. ( 점 두개 )를 사용한다
../images/이미지파일.jpg
.. -> 상위 폴더를 지칭하는 키워드
